### Description

    paramsDict: PyDictionary - (Dictionary) i.e. {'enabled':True, 'itemName': 'Bangers', 'itemPath':'FGBread%', 'itemId':1, 'itemIdList':[1,2,3], 'itemClassName': 'Bread%', itemClassId':1, 'itemClassIdList':[1,2,3]}


- **Returns**  
    (PyDataset) resultant query
